Custom Update Marker Example

Download Download View imgNotes on GitHub

This example demonstrates a custom implementation of the "onUpdateMarker" callback to change how the markers are rendered. In this case the size of the marker relative to the image is preserved. A custom "onAdd" callback inserts a circular div element with middle horizontal and vertical alignment (other alignment options wont work as expected). The "onUpdateMarker" callback changes the size of the the marker as the zoom changes.
